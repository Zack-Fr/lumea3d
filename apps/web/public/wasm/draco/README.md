# WASM Decoder Files

This directory contains the WASM decoder files for Draco compression.

## Required Files
- `draco_decoder.wasm` - Draco decoder WebAssembly binary
- `draco_wasm_wrapper.js` - Draco decoder JavaScript wrapper

## Installation
These files should be copied from the three.js examples directory:
```
node_modules/three/examples/jsm/libs/draco/
```

Or downloaded from the official Draco repository.
